# Sizing & Fit Guide

- Jackets and windbreakers run true to size. Order your usual size.
- Hiking boots run about half a size large; we recommend sizing down half a size.
- For layering in cold weather, consider sizing up one size on insulated jackets.
- A full size chart (cm and inches) is on every product page under the "Size & Fit" tab.
- If you are between sizes on apparel, size up for a relaxed fit or down for a slim fit.
- Unsure? Free size exchanges mean you can order two sizes and return the one that does not fit.
